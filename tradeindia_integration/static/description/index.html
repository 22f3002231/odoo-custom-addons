TradeIndia Integration

Automatically Fetch TradeIndia Leads into Odoo CRM

This module provides a complete integration with the TradeIndia API. It automates the process of fetching new business inquiries and creating them as qualified leads in your Odoo CRM, saving you time and ensuring no lead is missed.

Key Features

  • Automated lead fetching every 5 minutes
  • Manual fetch option for specific date ranges
  • Detailed mapping of all key lead data
  • Company, Address, and Contact Info capture
  • Prevents duplicate leads using unique ID
  • Creates entries as "Leads" for qualification
  • Built-in API Log Viewer for monitoring
  • Smart lead reactivation feature

How to Use

Configuration

1. Go to the TradeIndia → Configuration menu.
2. Enter your TradeIndia API credentials (User ID, Profile ID, API Key) and save.
3. Use the "Test Connection" button to ensure your credentials are valid.

Usage

The system will automatically fetch new leads every 5 minutes. To fetch leads from a past period, use the TradeIndia → Fetch Leads menu. To see a history of all API calls, visit the TradeIndia → API Logs menu.

Benefits

  • Never miss a business inquiry from TradeIndia
  • Save time with automated lead capture
  • Improve response time to potential customers
  • Streamline your sales qualification process
  • Track all API activity with detailed logs
  • Prevent duplicate lead entries automatically

Technical Details

API Integration: Uses TradeIndia's official inquiry API endpoint
Scheduling: Automated cron job runs every 5 minutes
Compatibility: Designed for Odoo 19 Community Edition
Security: API keys are stored encrypted in the database
